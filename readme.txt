An ebay auction display widget plugin, which displays scrolling auctions with thumbnails, customizable color styles and more.

== Description ==
Ebay auction scroller is an Ebay auction display widget. which displays a vertical scrolling Ebay auction listing filtered by region, category and seller id.
It uses the jQuery easy ticker plugin to add a news ticker like effect to the auction feeds.
Requires Ebay developer application id to function. To obtain an app id visit https://go.developer.ebay.com/

The plugin is fully customizable with external styles and with some inbuilt color styles.


= Features =

* jQuery [news ticker like effect](http://www.aakashweb.com/jquery-plugins/easy-ticker/) to the post feeds (can turn off or on)
* Displays thumbnail to the feed items if available.
* Customizable with Inbuilt color styles and with external CSS styles.
* Customizable ticker speed.
* Can tweak the widget to change the no of visible feed items and more...


== Installation ==

Download the latest version of Ebay auction scroller

1. Unzip & upload it to your WordPress site.
1. Activate the plugin.
1. Drag and drop the "Ebay auction scroller" widget in the "Widgets" page.
1. Enter your Ebay developer app id. Note, anything can be entered here to activate the options. However without a valid ebay developer app id The widget will not function.
1. Choose the locale and categories to display in the widget, tweak some settings and you are,
1. Done !

== Frequently Asked Questions ==

= How can I customize the Vertical Post Scroller widget externally? =

You can use the `Ebay auction scroller` class in your stylesheet to control the widget styling. Other classes are,

`wi-tab-wrap` - the tab's class.
`wi-wrap` - the wrapper of the widget.
`wi-item.odd` - to control the odd feed items.
`wi-item.even` - to control the even feed items.

= Can I use the widget without an Ebay developer application id? =

No, sorry. With some hacking you could make it scroll through your blog posts ;-)

= How can I use my ebay affiliate membership to earn commision? =

You will have to buy the pro version which is at present in the final stages of development. This widget is only free to those who sell their own product.
